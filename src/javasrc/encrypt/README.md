### Encrypt
信息加密的理解和应用

##### 对称加密:  
明文<->秘钥<->密文  
依赖于秘钥的安全/得知任意其二可破解其余的一项  

##### 非对称加密传输:  
A:请求公钥  
B:wqe123!@#  
A:xxxxxxxx（公钥加密的明文）  
B:用私钥解密 xxxxxxxx  


PS:  
对非对称加密的解密慢 所以可以非对称加密<对称加密的秘钥>
